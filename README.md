# 💳 Secure Credit Card Issuer Back Office System

This repository contains the full implementation and documentation of the **Secure Credit Card Issuer Back Office System**, developed by **Lew Yu Tian** as part of the UTAR Final Year Project (FYP).

---

## 🏗️ Project Overview
The system provides a secure and efficient platform for managing **credit card issuing operations** through role-based access control (RBAC) and maker-checker approval workflows

---

## 🧩 System Architecture
**Frontend:** Angular (issuer-frontend)  
**Backend:** Spring Boot (spring-backend)  
**Database:** PostgreSQL
**Version Control:** Git & GitHub

---

## ⚙️ Features
1. **Role-Based Access Control (RBAC)** — Three distinct roles: Admin, Manager (Checker), and Staff (Maker).  
2. **Maker-Checker Workflow** — Two-step approval for critical operations such as account creation and card issuance.  
3. **Customer, Account & Card Modules** — Complete CRUD functionality for managing issuer operations.
